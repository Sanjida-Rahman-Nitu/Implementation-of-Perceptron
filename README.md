The Perceptron is one of the earliest binary classification algorithms.

It is a linear classifier that:
- Takes input features
- Applies weights and bias
- Uses a step activation function
- Outputs class 0 or 1

    y = step(w Â· x + b)

Where:
- w = weights
- x = input features
- b = bias
